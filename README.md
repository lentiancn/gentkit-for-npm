## 介绍

打造一组可以最少依赖、干净的开源框架，支持每个组件、文件甚至代码段可以独立使用，相互没有耦合。

## 开发语言

- Java JDK 17，但是如果需要应用旧版本环境，请移步至 [Java JDK 1.8](https://gitee.com/gentkit/gentkit-for-java-jdk8)
